¿Qué es Java y cómo funciona?

Java es un lenguaje de programación desarrollado por Sun Microsystems en 1995 y ahora es mantenido por Oracle. Es conocido por su versatilidad, seguridad y portabilidad.

JDK (Java Development Kit)

El JDK es un paquete de herramientas proporcionado por Oracle que contiene todo lo necesario para desarrollar aplicaciones en Java.

Contenido del JDK:

  • JRE (Java Runtime Environment):
    • JVM (Java Virtual Machine): Ejecuta el bytecode (.class).
    • Bibliotecas de clases:
      • java.lang (clases básicas como String, System).
      • java.util (colecciones, fechas).
      • java.io (manejo de archivos).
      • java.net (conexiones de red).
  • Herramientas de desarrollo:
    • javac: Compilador (.java -> .class).
    • java: Ejecutador de aplicaciones.
    • javadoc: Genera documentación HTML.

Tipos de Datos

En los fundamentos de programación tenemos tipos de datos primitivos junto a sus wrappers y los no primitivos. Los tipos de datos son una categorización de los valores que puede almacenar una variable y las operaciones que se pueden hacer con ese tipo de dato.

Datos Primitivos

Los datos primitivos en java son los que almacenan un valor en memoria, están almacenadas en el Stack (memoria rápida para datos efímeros). No pertenecen a un objeto, no contienen métodos ni atributos y son medibles (0, 0.0, false).

Datos Wrapper

En Java, los tipos de datos wrapper pertenecen a clases dedicadas a convertir tipos de datos primitivos a no primitivos. Se convierten principalmente para poder hacer operaciones de métodos y aplicarle lógica con valores primitivos.

Datos no primitivos (Referenciados)

Son objetos que almacenan una referencia (dirección de memoria) al dato real que está en el heap. Pueden ser null, tienen métodos y atributos.

Variables y Palabras Reservadas

¿Qué es una variable?

En programación una variable es un espacio en memoria el cual contiene un único dato. Con ese dato asignado a la variable se pueden hacer múltiples operaciones según la lógica programada.

¿Qué es una palabra reservada?

Una palabra reservada o una keyword es un término especial definido por el lenguaje que tiene un significado específico y no puede ser usado como identificador (nombre de variable, método, clase etc.).

Operadores

En Java, una expresión es una combinación de variables, operadores, literales y llamadas a métodos que, al ser evaluada, produce un valor único.

Operadores Aritméticos

Se basan en las operaciones normales como suma, resta, multiplicación, división y resto (módulo).

Operadores de Asignación Compuesta

Combinan una operación aritmética con una asignación en una sola expresión, permitiendo modificar el valor de una variable de forma simplificada (ej. x += 5).

Operadores Relacionales

Son símbolos que comparan dos valores y devuelven un resultado booleano (true o false), como == (igual a) o > (mayor que).

Operadores Lógicos

Permiten combinar o modificar expresiones condicionales. Los principales son && (AND), || (OR) y ! (NOT).

Condicionales (Estructuras de Control)

En Java, una condición es una expresión lógica (que evalúa a true o false) que determina el flujo de ejecución de un programa. A través de estructuras de control como if-else, switch o el operador ternario, el programa decide qué bloque de código ejecutar.

Estructura if-else

Sirve para determinar si una condición se cumple o no. Si la condición del if es verdadera se ejecuta su bloque, de lo contrario, se ejecuta el bloque del else.

Estructura switch

Es un bloque de control que se utiliza para ejecutar uno o varios bloques de código basándose en el valor de una expresión, siendo una estructura multidireccional.

Bucles (Estructuras de Repetición)

Los bucles son bloques de control que permiten ejecutar un mismo código múltiples veces mientras se cumpla una condición. Son fundamentales para automatizar tareas repetitivas.

Estructura while

Ejecuta un bloque de código mientras una condición es verdadera. La condición se evalúa antes de cada iteración.

Bucle do-while

Ejecuta un bloque de código al menos una vez, y luego repite la ejecución mientras una condición sea verdadera. La condición se evalúa después de la primera iteración.

Bucle for

Es una estructura determinista diseñada para ejecutar un bloque de código un número específico de veces, con control sobre la inicialización, condición y actualización.

Arrays (Arreglos)

Los Arrays son estructuras de datos que permiten almacenar múltiples valores del mismo tipo en una sola variable. Son de tamaño fijo, es decir, una vez definidos, no pueden crecer o reducirse dinámicamente.

Sintaxis Básica

  • Declaración: tipo[] nombreArray;
  • Inicialización con tamaño: nombreArray = new tipo[tamaño];
  • Inicialización con valores: tipo[] nombreArray = {valor1, valor2};
